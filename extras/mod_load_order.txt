How to Load Mods
=================

Using ZDL, Make It So or similar launcher:
1. Add each .pk3 in the order listed above.
2. Select your IWAD (Doom II or Ultimate Doom).
3. Launch GZDoom.

Using terminal / command line:
gzdoom -iwad doom2.wad -file BRUTALPACK\ 10.4.pk3 Brutal\ Pack\ Neural\ Pack.pk3 ... behold.pk3

Replace behold.pk3 with your preferred invincibility version, e.g.:
- behold_inv_3.pk3 for 3 seconds
- behold_inv_5.pk3 for 5 seconds
- behold_inv_10.pk3 for extended safety

Only load one of the behold_inv_#.pk3 files at a time!
